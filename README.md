# :memo: Prerequisites:
- Nodejs (≥19.1.0)
- Rust (≥1.66.1)

# :warning: Getting started:
- Install Nodejs and Rust.
- install depencies using npm:
  ```
  npm install
  ```

# :information_source: Development:
- Runing the app:
  ```shell
  npm run tauri dev
  ```
- Building the app:
  ```shell
  npm run tauri build
  ```
