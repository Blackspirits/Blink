# 📝 Prerequisites:
- Nodejs (≥19.1.0)
- Rust (≥1.66.1)

# ℹ️ Getting started:
- Install Nodejs and Rust.
- install depencies using npm:
  ```shell
  npm install
  ```

# 🛠️ Development:
- Running the app:
  ```shell
  npm run tauri dev
  ```
- Building the app:
  ```shell
  npm run tauri build
  ```
- other commands can be found inside the `"scripts"` inside [package.json](https://github.com/prayag17/JellyPlayer/blob/main/package.json)
